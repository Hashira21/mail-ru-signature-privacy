Политика конфиденциальности

Дата последнего обновления: 3 октября 2025 г.

Краткая версия: Расширение хранит ваши данные локально в браузере и НЕ передает их на внешние серверы. Ваша конфиденциальность полностью защищена.

1. Введение

Расширение "Mail.ru Signature Automator" разработано для автоматической вставки корпоративных подписей в почтовый сервис Mail.ru для сотрудников компании MARKS Group. Мы серьезно относимся к защите вашей конфиденциальности.

2. Собираемые данные

Расширение сохраняет следующую информацию локально в вашем браузере:

  • Личные данные: фамилия, имя, отчество
  • Рабочая информация: должность, отдел
  • Контактные данные: телефон, email
  • Местоположение: выбранный город офиса
  • Технические данные: последние введенные значения для удобства повторного использования

3. Как мы используем данные

Собранные данные используются исключительно для:

  • Генерации HTML-кода email-подписи
  • Автоматической вставки подписи в редактор Mail.ru
  • Сохранения ваших предпочтений для последующего использования

4. Хранение данных

Важно: Все данные хранятся локально в браузере с использованием Chrome Storage API.

  • Данные НЕ передаются на внешние серверы
  • Данные НЕ отправляются разработчикам
  • Данные НЕ передаются третьим лицам
  • Данные остаются только на вашем компьютере

5. Разрешения расширения

Расширение запрашивает следующие разрешения:

  • activeTab: для взаимодействия с активной вкладкой Mail.ru
  • storage: для локального сохранения введенных данных
  • host_permissions (e.mail.ru): для работы только на сайте Mail.ru

Мы НЕ запрашиваем доступ к:

  • Истории браузера
  • Закладкам
  • Другим сайтам
  • Вашим паролям
  • Содержимому email

6. Безопасность

Расширение использует только безопасные API браузера Chrome. Код расширения не содержит:

  • Вредоносных скриптов
  • Трекеров и аналитики
  • Рекламы
  • Подключений к внешним серверам

7. Удаление данных

Вы можете в любой момент:

  • Удалить расширение из браузера - все данные будут автоматически удалены
  • Очистить данные расширения через настройки Chrome: chrome://settings/content/siteDetails?site=chrome-extension://[ID расширения]

8. Изменения в политике

При внесении существенных изменений в политику конфиденциальности мы обновим дату в начале документа. Рекомендуем периодически проверять эту страницу.

  • Email: privacy@marksgroup.ru
